Removal
NOTES:

  • This procedure contains some variation in the illustration depending on the vehicle specification, but essential information is always correct.

  • This procedure contains illustrations showing certain components removed to provide extra clarity.


  1. Move the front row seat forward.
  2. Remove the front row seatbelt anchor cover.
    • Remove and discard the anchor bolt.
    • Release the front row seatbelt lower anchor.
  4. Move the front row seat backrest to the lowest position.
  5. Release the 2 door aperture seals.
    1. Release the 2 lower plastic clips.
    1. Release the 2 center clips.
  7. CAUTION:

    Failure to follow this instruction may cause damage to the vehicle.

    1. Move the upper trim panel from the B-pillar as illustrated.
    1. Release the 2 upper locating legs.
    1. Remove the B-pillar upper trim panel.
Installation
  1. CAUTION:

    Make sure the locating legs are installed correctly.

    1. Install the B-pillar upper trim panel.
    1. Secure the 2 center clips.
    1. Secure the 2 lower plastic clips.
  2. Install the 2 door aperture seals.
  3. CAUTION:

    Make sure that the new bolt is installed.

    • Install the front row seatbelt lower anchor.
    • Install and tighten the bolt.
      Renew Part: Seatbelt anchor bolt Quantity: 1.
      Torque: 40 Nm
  4. Install the front row seatbelt anchor cover.
  5. Move the front row seat to the original position.
