Removal
NOTE:
  • This procedure contains some variation in the illustrations depending on the vehicle specification, but the essential information is always correct.
  • This procedure contains illustrations showing certain components removed to provide extra clarity.

  1. Raise and support the vehicle on a suitable 2 post lift.
    Refer to: Lifting (100-02 Jacking and Lifting, Description and Operation).
  2. Remove the rear halfshaft
    Refer to: Rear Halfshaft (205-05 Rear Drive Halfshafts, Removal and Installation).
  3. Remove and discard the 2 retaining clips.
  4. CAUTION:

    Use suitable protective covers to protect the halfshaft.

    Position the halfshaft in a vice as illustrated.
  5. Remove the inner constant velocity joint using a soft faced hammer.
  6. NOTE:

    Note the orientation of the component prior to removal.

    Using a suitable tool mark the position of the component.
  7. Remove and discard the retaining clips.
  8. NOTE:

    Clean all the mating faces and reusable parts thoroughly and check for damage.

    • Remove the bearings using a suitable puller.
      Special Tool(s): 303-D121
  9. CAUTION:

    Discard the component.

    Remove and discard the inner Constant Velocity (CV) joint boot.
Installation
  1. Install the Constant Velocity (CV) joint boot and retaining clip.
    Renew Part: Inner constant velocity joint boot kit Quantity: 1.
  2. NOTE:

    Make sure that the component is installed to the position noted on removal.

    Install a new retaining clip.
    Renew Part: Circlip Quantity: 1.
  3. Install the Inner CV joint.
    • Pack the inner CV joint with the grease supplied.
    • Install the 2 retaining clips.
  5. Install the rear halfshaft.
    Refer to: Rear Halfshaft (205-05 Rear Drive Halfshafts, Removal and Installation).
