Fuel System Bleeding (G1801385)

GENERAL PROCEDURES

Loading...

WARNING:

After carrying out repairs, the fuel system must be checked visually for leaks. Failure to follow this instruction may result in personal injury.

NOTE:

This procedure is necessary if any low-pressure fuel system components are removed or replaced. These include the fuel filter, fuel lines, fuel tank or fuel cooler.


  1. CAUTION:

    Do not start the engine.

    1. Set the ignition to the ON position.
    1. Wait for 65 seconds.
    1. Set the ignition to the OFF position.
    1. Repeat the above steps 2 more times.
  2. Start the engine and idle for 60 seconds
