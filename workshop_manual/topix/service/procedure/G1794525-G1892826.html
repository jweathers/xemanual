Installation
CAUTION:

Where structural adhesive is to be applied between surfaces it is essential compression is applied to the joints during the curing period.

NOTES:

  • BSF installation involves using the Jaguar approved BSF installation tool and requires a hole diameter of 6.5mm.

  • SPR installation requires using the Jaguar approved SPR tool.

  1. Dress the flanges where necessary.
  2. Detach the luggage compartment lid damper bracket from the new back panel outer service assembly as indicated.
  3. NOTE:

    Make sure that the edges of the rear floor side extension are repositioned.

    Offer up the back panel outer and clamp into position. Check alignment if correct proceed to next step, if not rectify and recheck before proceeding.
  4. Drill holes where the metal inert gas (MIG) plug welds and the BSF's are to be installed as indicated.
  5. Remove the back panel outer.
  6. Deburr the drilled holes.
  7. Clean and prepare the panel surfaces.
  8. Apply the coupling agent where the Jaguar recommended bonding material is to be applied and allow to dry.
  9. NOTES:

    • Make sure a continuous bead of adhesive surrounds fixing holes.

    • Do not apply adhesive in the vicinity of the MIG plug welds.

    Apply a 5mm zig zag bead of the Jaguar recommended bonding material as indicated.
  10. Offer up the back panel outer and clamp into position. Check alignment if correct proceed to next step, if not rectify and recheck before proceeding.
  11. Offer up the luggage compartment lid damper bracket and clamp into position. Check alignment if correct proceed to next step, if not rectify and recheck before proceeding.
  12. Install the spot welds as indicated.
  13. Install the BSF's and the SPR's as indicated.
  14. Install the spot and MIG plug welds as indicated.
  15. Install the spot and MIG plug welds as indicated.
  16. Dress the welds.
  17. Remove any excess adhesive.
  18. Make sure that any open or exposed panel joints are correctly sealed.
  19. Make sure corrosion protection is applied to all areas affected by repair.
  20. The installation of associated panels and components is the reversal of removal procedure.
