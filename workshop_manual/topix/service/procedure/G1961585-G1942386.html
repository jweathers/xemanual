Configure vehicle to the market; set customer options

Configure the vehicle to the market using the Instrument cluster via the steering wheel controls and Touch Screen.

NOTE:

the vehicle may power down after three minutes. If this happens, switch the ignition back ON and continue configuring to Market.

Configure the Touch Screen options

Set the Clock, Date, Language, and the Unit of measurement specific to the Market unless otherwise instructed.

Clock - with InControl® Touch™ only

  • Select general settings > System settings > Time and date.
    • Set the correct time.

Date

  • Select general settings > System settings > Time and date.
    • Set the correct date.

Language

  • Select general settings > System settings > Language.
    • Set the language.

Unit of measurement

  • Select general settings > System settings > Unit of measurement.
    • Set the Unit of measurement.

Auto High Beam (AHB)

  • For information: the automatic headlamp system has an additional feature called Auto High Beam. The ‘Hand of Traffic’ must be set in the Message Centre and the feature enabled to make sure of correct operation of the system.
  • Set the Auto High Beam (AHB).
  • Select Driving features > Auto high beam.
  • Configure the ‘Hand of Traffic’ setting by selecting Drive on Right (of road) ONLY.
  • Select Activate Assist.
  • Carry out a visual check to make sure that there are no obstructions directly in view of the HBA camera sensor.
NOTE:

The AHB feature can be enabled (or disabled) by selecting (or de-selecting) Activate Assist.

Enabling or disabling AHB will not affect previous ‘Hand of Traffic’ settings.

Easy Entry/Exit

  • Select Vehicle Set-up > Easy Entry/Exit
    • Before the road test, make sure that when entering the vehicle the seat moves back and the steering wheel moves up to ease entry. The seat and steering wheel should return to the normal position when the ignition is switched on. The reverse should happen when exiting the vehicle.

2 Stage Vehicle Locking

  • Select Vehicle Set-up > Single Point Entry Vehicle Locking

Drive-Away Door Locking

  • Select Vehicle Set-up > Drive-Away Door Locking
    • During the road test make sure that all the doors lock when the vehicle speed exceeds 8 km/h (5 mph).

Reverse-dip Mirror

  • Select Vehicle Set-up > Reverse-dip Mirror
    • During the road test make sure that when selecting reverse the mirrors dip accordingly.

Wiper rain sensor

  • Select Vehicle Set-up > Wiper rain sensor

Bluetooth®

Using a Bluetooth® capable phone:

NOTE:

Make sure that the device is enabled for Bluetooth®.

Please refer to the Owners section of the Jaguar website at www.Jaguar.com, for a list of compatible phones.

There are two ways of pairing the Bluetooth® device and the vehicle. Detailed below is pairing to the vehicle, the other method is using the 'Vehicle to device' to pair to the device. If you use the 'Vehicle to device' method, follow the instructions on screen.

  1. Switch the ignition ON.
  2. Switch the Touch Screen ON.
  3. Press the General settings button.
  4. Select Bluetooth® from the displayed menu options.
  5. From the list, select Make system discoverable.
  6. Switch on device’s Bluetooth® wireless technology device connection and search for Bluetooth® devices.
    • On some phones, this is referred to as 'new paired device'. See phone’s operating instructions for further information.
  7. A Passkey number will appear on the phone.
    • If this number matches the number on the Touch Screen , select Yes or press Pair on the phone.

Once a phone is paired, it will appear on the connected device list and Phone screen.

Make sure the front and rear electric windows operate correctly from the drivers door controls and each of the passenger doors. Check the manual and one-touch operation.

Make sure the roof opening panel and roof opening panel blind operate correctly (if equipped).

Make sure the rear windshield sunblind operates correctly.

Interior and exterior lights

Check all interior lights (except the glovebox and vanity mirror lamps) illuminate when the vehicle is unlocked or a door is opened.

Check the operation of all exterior lights.

Seatbelts and seat functions

Check all seatbelts are secure at the mounting points, lock and release from the buckle correctly and the belt locks if quickly pulled. The belt should retract to the original position.

Check the front seat positions can be adjusted via the front seat control switch (if equipped).

Disconnect the battery power supply from the vehicle.

Interior damage and cleanliness

Remove all interior protection and make sure that the interior of the vehicle is undamaged.

Make sure the HUD display protective cover is removed (if equipped).

Make sure the front door trim protection fixing velcro tags are removed. Pull the four fixing velcro tags to remove and discard.

Record any damage on the PDI form.
