Roof Sheet Metal Repairs

Front Roof Reinforcement Panel (G2294714)

REMOVAL AND INSTALLATION

Loading...

Removal
NOTE:

In this procedure the front roof reinforcement panel is installed in conjunction with:

  • Roof panel.

  1. The front roof reinforcement panel is serviced as indicated.
  2. Before commencing this procedure make sure that you are aware of all Health and Safety requirements.
    Refer to: Health and Safety Precautions (100-00 General Information, Description and Operation).
  3. NOTES:

    • Where Breakstem Fastener (BSF) removal is instructed remove the fastener shaft with a 4 mm punch and remove the head with a 6.5 mm drill bit. Clean all the drilled holes to make sure they are free from debris.

    • Where Self Piercing Rivet (SPR) removal is instructed use either the Jaguar Land Rover (JLR) approved SPR tool or a 5.3 mm drill bit. Clean all the drilled holes to make sure they are free from debris.

    • Any joints sealed as part of vehicle manufacture must be sealed as part of the repair.

    When installing the front roof reinforcement panel make sure that you are aware of the methods, tools and fixings used in this procedure.
    Refer to: Body Repairs (501-25A Body Repairs - General Information, Description and Operation).
  4. For additional information
    Refer to: Roof Panel - Vehicles With: Roof Opening Panel (501-28 Roof Sheet Metal Repairs, Removal and Installation).
    Refer to: Roof Panel - Vehicles Without: Roof Opening Panel (501-28 Roof Sheet Metal Repairs, Removal and Installation).
  5. Release and position the wiring harness to one side.
  6. Remove the self piercing rivets as indicated.
  7. Carefully separate the joints and remove the front roof reinforcement panel.
Installation
CAUTION:

Where structural adhesive is to be applied between surfaces it is essential compression is applied to the joints during the curing period.

NOTES:

  • Breakstem Fastener (BSF) installation involves using the Jaguar Land Rover (JLR) approved BSF installation tool and requires a hole diameter of 6.5 mm.

  • Self Piercing Rivet (SPR) installation requires using the JLR approved SPR tool.


  1. Remove the rivet remnants.
  2. Dress the flanges where necessary.
  3. Offer up the new front roof reinforcement panel and clamp into position. Check alignment, if correct proceed to next step, if not rectify and check the alignment again before proceeding.
  4. Drill holes where the Breakstem Fasteners (BSFs) are to be installed as indicated.
  5. Remove the new front roof reinforcement panel.
  6. Deburr the drilled holes.
  7. Using a fine bristle disc, clean and prepare the panel surfaces.
  8. Apply the coupling agent where the Jaguar Land Rover (JLR) recommended bonding material is to be applied and allow to dry.
  9. NOTE:

    Make sure a continuous bead of adhesive surrounds fixing holes.

    Apply a 5 mm zig zag bead of the JLR recommended bonding material as indicated.
  10. Offer up the new front roof reinforcement panel and clamp into position. Check alignment, if correct proceed to next step, if not rectify and check the alignment again before proceeding.
  11. Install the BSFs from the inside as indicated.
  12. Remove any excess adhesive.
  13. Make sure that all installed BSFs and SPRs are sealed using a JLR recommended sealer.
  14. Make sure that any open or exposed panel joints are correctly sealed.
  15. Make sure corrosion protection is applied to all areas affected by the repair.
  16. The installation of associated panels and components is the reversal of the removal procedure.
