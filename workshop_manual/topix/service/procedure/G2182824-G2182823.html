Removal

  1. The side panel reinforcement rear extension is serviced as indicated.
  2. Before commencing this procedure make sure that you are aware of all Health and Safety requirements.
    Refer to: Health and Safety Precautions (100-00 General Information, Description and Operation).
  3. For further information on the methods, tools and fixings used in this procedure refer to the body repairs - general information section.
    Refer to: Body Repairs (501-25A Body Repairs - General Information, Description and Operation).
    4. NOTES:

    • Where Break Stem Fasteners (BSF) removal is instructed remove fastener shaft with a 4mm punch and remove the head with a 6.5 mm drill bit. Clean all the drilled holes to make sure they are free from debris.

    • Where Self Piercing Rivet (SPR) removal is instructed use either the approved SPR tool or 5.3 mm drill bit.

    • Any joints sealed as part of vehicle manufacture must be sealed as part of the repair.

Installation
CAUTION:

Where structural adhesive is to be applied between surfaces it is essential compression is applied to the joints during the curing period.

NOTES:

  • Break Stem Fastener (BSF) installation involves using the Jaguar Land Rover (JLR) approved BSF installation tool and requires a hole diameter of 6.5 mm.

  • Self Piercing Rivet (SPR) installation requires using the JLR approved SPR tool.


  1. Remove the rivet remnants.
  2. Dress the flanges where necessary.
  3. Offer up the new side panel reinforcement rear extension and clamp into position. Check alignment, if correct proceed to next step, if not rectify and check the alignment again before proceeding.
  4. Drill holes where the BSFs are to be installed as indicated.
  5. Remove the new side panel reinforcement rear extension.
  6. Deburr the drilled holes.
  7. Using a fine bristle disc clean and prepare panel surfaces.
  8. Apply the coupling agent where the JLR recommended bonding material is to be applied and allow to dry.
    9. NOTE:

    Make sure a continuous bead of adhesive surrounds fixing holes.

  9. Apply a 5 mm zig zag bead of the JLR recommended bonding material as indicated.
  10. Offer up the new side panel reinforcement rear extension and clamp into position. Check alignment, if correct proceed to next step, if not rectify and check the alignment again before proceeding.
  11. Make sure the side panel reinforcement rear extension is clamped in position.
  12. Install the BSFs as indicated
  13. Install the SPRs as indicated.
  14. Remove any excess adhesive.
  15. Make sure that any open or exposed panel joints are correctly sealed.
  16. Make sure that all installed BSFs and SPRs are sealed using a JLR recommended sealer.
  17. Make sure corrosion protection is applied to all areas affected by the repair.
  18. The installation of associated panels and components is the reversal of the removal procedure.
