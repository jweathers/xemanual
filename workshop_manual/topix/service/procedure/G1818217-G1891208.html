Removal
NOTES:

  • This procedure contains some variation in the illustrations depending on the vehicle specification, but the essential information is always correct.

  • This procedure contains illustrations showing certain components removed to provide extra clarity.


  1. Raise and support the vehicle on a suitable 2 post lift.
    Refer to: Lifting (100-02 Jacking and Lifting, Description and Operation).
  2. Remove the rear subframe.
    Refer to: Rear Subframe (502-00 Uni-Body, Subframe and Mounting System, Removal and Installation).
    • Remove and discard the 4 rear stabilizer bar bracket bolts.
    • Remove and discard 2 the rear stabilizer bar link nuts.
    • Remove the rear stabilizer bar from the rear subframe.
    • Remove the rear stabilizer bar brackets.
    • Remove the rear stabilizer bar bushes.
Installation
    • Install the rear stabilizer bar bushes.
    • Install the rear stabilizer bar brackets.
    • Install the rear stabilizer bar to the rear subframe.
    • Install and tighten 4 new rear stabilizer bar bracket bolts.
      Renew Part: Rear stabilizer bar to subframe bolts Quantity: 4.
      Torque: 60 Nm
    • Install and tighten 2 new rear stabilizer bar link nuts.
      Renew Part: Stabilizer bar link upper nut Quantity: 2.
      Torque: 48 Nm
  3. Install the rear subframe.
    Refer to: Rear Subframe (502-00 Uni-Body, Subframe and Mounting System, Removal and Installation).
