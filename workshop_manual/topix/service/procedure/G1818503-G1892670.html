Removal
CAUTION:

Protect the surrounding paintwork to avoid damage.

NOTES:

  • This procedure contains illustrations showing certain components removed to provide extra clarity.

  • This procedure contains some variation in the illustrations depending on the vehicle specification, but the essential information is always correct.

  • Make sure the door window is in the fully closed position.


  1. Disconnect the startup battery ground cable.
    Refer to: Startup Battery Disconnect and Connect (414-01 Battery, Mounting and Cables, General Procedures).
  2. To avoid accidental deployment, wait at least 2 minutes after you disconnect the startup battery ground cable.
  3. Remove the exterior front door handle.
    Refer to: Exterior Front Door Handle (501-14 Handles, Locks, Latches and Entry Systems, Removal and Installation).
  4. Remove the 3 bolts.
    • Disconnect the 2 electrical connectors.
    • Release the wiring harness clip.
    • Remove the screw.
    • Remove the gasket.
  7. Remove the front door latch assembly.
  8. NOTE:

    Only do the following step(s) if a new component is to be installed.

    Release the 2 boots.
  9. Remove the upper cover.
  10. Remove the lower cover.
    • Detach the exterior handle cables from the door latch.
    • Detach the door lock cables from the door latch.
  12. Detach the front door interior handle release cable.
Installation
  1. Attach the front door interior handle release cable.
    • Attach the door lock cables to the door latch.
    • Attach the exterior handle cables to the door latch.
  3. Install the lower cover.
  4. Install the upper cover.
  5. Install the 2 boots.
  6. Install the front door latch assembly.
    • Install and tighten the screw.
      Torque: 3.2 Nm
    • Install the gasket.
    • Connect the 2 electrical connectors.
    • Install the wiring harness clip.
  9. Install and tighten the 3 bolts.
    Torque: 7 Nm
  10. Install the exterior front door handle.
    Refer to: Exterior Front Door Handle (501-14 Handles, Locks, Latches and Entry Systems, Removal and Installation).
  11. Connect the startup battery ground cable.
    Refer to: Startup Battery Disconnect and Connect (414-01 Battery, Mounting and Cables, General Procedures).
