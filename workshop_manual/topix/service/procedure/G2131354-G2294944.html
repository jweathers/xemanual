Removal
WARNING:

Be prepared to collect escaping oil.

NOTES:

  • This procedure contains some variation in the illustration depending on the vehicle specification, but essential information is always correct.

  • This procedure contains illustrations showing certain components removed to provide extra clarity.


  1. Remove the engine cover.
    Refer to: Engine Cover - GTDi 2.0L Petrol (501-05 Interior Trim and Ornamentation, Removal and Installation).
    Refer to: Engine Cover - INGENIUM I4 2.0L Diesel (501-05 Interior Trim and Ornamentation, Removal and Installation).
    Refer to: Engine Cover - V6 S/C 3.0L Petrol (501-05 Interior Trim and Ornamentation, Removal and Installation).
    Refer to: Engine Cover - GTDi 2.0L Petrol (501-05 Interior Trim and Ornamentation, Removal and Installation).
  2. Release the Emergency Park Release (EPR).
  3. Raise and support the vehicle on a suitable 2 post lift.
    Refer to: Lifting (100-02 Jacking and Lifting, Description and Operation).
    General Equipment: 2 post lift
  4. Drain the rear differential oil.
    Refer to: Differential Draining and Filling vehicles without heavy duty differential (205-02 Rear Drive Axle/Differential, General Procedures).
  5. Remove the exhaust system.
    Refer to: Exhaust System (309-00A Exhaust System - GTDi 2.0L Petrol, Removal and Installation).
    Refer to: Exhaust System - Vehicles Without: Diesel Exhaust Fluid (309-00B Exhaust System - INGENIUM I4 2.0L Diesel, Removal and Installation).
    Refer to: Exhaust System - Vehicles With: Diesel Exhaust Fluid (309-00B Exhaust System - INGENIUM I4 2.0L Diesel, Removal and Installation).
    Refer to: Exhaust System - Vehicles With: Diesel Exhaust Fluid (309-00B Exhaust System - INGENIUM I4 2.0L Diesel, Removal and Installation).
    Refer to: Exhaust System (309-00C Exhaust System - V6 S/C 3.0L Petrol, Removal and Installation).
  6. Remove the 2 rear halfshafts.
    Refer to: Rear Halfshaft (205-05 Rear Drive Halfshafts, Removal and Installation).
    • Mark the position of the driveshaft in relation to the differential drive flange.
    • Remove and discard the 6 driveshaft to differential drive flange bolts.
    • Remove the 3 washers.
  8. CAUTION:

    Make sure the tool is correctly installed in the groove on the driveshaft.

    Install the special tool and release the driveshaft from the rear differential drive flange.
    Special Tool(s): 205-932
  9. NOTE:

    If equipped.

    Disconnect the oil temperature sensor.
  10. NOTE:

    If equipped.

    Disconnect the breather and the locking motor electrical connector.
  11. Remove the 2 nuts and the vibration damper.
    • Position a suitable transmission jack below the rear differential case.
      General Equipment: Transmission jack
    • Remove the 2 rear differential case mounting bolts.
  13. Remove the 2 front differential case mounting bolts and remove the differential case.
  14. Remove and discard the breather.
Installation
  1. Install a new breather.
    Renew Part: Breather Quantity: 1.
  2. Install the differential case and hand tighten the 2 front mounting bolts.
  3. Install and tighten the 2 rear differential case mounting bolts.
    Torque: 192 Nm
  4. Tighten the 2 front mounting bolts.
    Torque: 110 Nm
    • Install the vibration damper.
    • Install and tighten the 2 nuts.
      Torque: 24 Nm
    • Install the rear drive shaft to the differential drive flange in the marked position.
    • Install the 3 washers.
    • Install and tighten 6 new bolts.
      Renew Part: Driveshaft to rear differential bolts Quantity: 6.
      Torque: 75 Nm
  7. NOTE:

    If equipped.

    Connect the oil temperature sensor.
  8. NOTE:

    If equipped.

    Connect the breather and the locking motor electrical connector.
  9. Install both the rear halfshafts.
    Refer to: Rear Halfshaft (205-05 Rear Drive Halfshafts, Removal and Installation).
  10. Install the exhaust system.
    Refer to: Exhaust System (309-00A Exhaust System - GTDi 2.0L Petrol, Removal and Installation).
    Refer to: Exhaust System - Vehicles Without: Diesel Exhaust Fluid (309-00B Exhaust System - INGENIUM I4 2.0L Diesel, Removal and Installation).
    Refer to: Exhaust System - Vehicles With: Diesel Exhaust Fluid (309-00B Exhaust System - INGENIUM I4 2.0L Diesel, Removal and Installation).
    Refer to: Exhaust System - Vehicles With: Diesel Exhaust Fluid (309-00B Exhaust System - INGENIUM I4 2.0L Diesel, Removal and Installation).
    Refer to: Exhaust System (309-00C Exhaust System - V6 S/C 3.0L Petrol, Removal and Installation).
  11. Fill the rear differential oil.
    Refer to: Differential Draining and Filling vehicles without heavy duty differential (205-02 Rear Drive Axle/Differential, General Procedures).
  12. Reset the Emergency Park Release (EPR).
  13. Install the engine cover.
    Refer to: Engine Cover - GTDi 2.0L Petrol (501-05 Interior Trim and Ornamentation, Removal and Installation).
    Refer to: Engine Cover - INGENIUM I4 2.0L Diesel (501-05 Interior Trim and Ornamentation, Removal and Installation).
    Refer to: Engine Cover - V6 S/C 3.0L Petrol (501-05 Interior Trim and Ornamentation, Removal and Installation).
    Refer to: Engine Cover - GTDi 2.0L Petrol (501-05 Interior Trim and Ornamentation, Removal and Installation).
