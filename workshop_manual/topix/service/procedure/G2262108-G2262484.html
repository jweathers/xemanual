Removal
NOTES:

  • This procedure contains some variation in the illustrations depending on the vehicle specification, but the essential information is always correct.

  • This procedure contains illustrations showing certain components removed to provide extra clarity.


  1. Remove the windshield wiper pivot arm.
    Refer to: Windshield Wiper Pivot Arm (501-16 Wipers and Washers, Removal and Installation).
    • Remove the 2 clips.
    • Release the seals.
  3. Disconnect the windshield washer hose.
  4. Remove the right access cover.
  5. Remove the left access cover.
  6. Disconnect the electrical connector at the right side.
  7. Disconnect the electrical connector at the left side.
    • Remove the 12 clips.
    • Remove the cowl panel.
  9. NOTES:

    • Only do the following step(s) if a new component is to be installed.

    • Repeat the step for other side.

    • Disconnect the windshield washer hose from the heated windshield washer jet.
    • Release the 3 clips.
    • Remove the heated windshield washer jet from the cowl panel.
    • Release the 8 clips.
    • Remove the windshield washer hose assembly from the cowl panel.
Installation
  1. NOTE:

    Only do the following step(s) if the component was previously removed.

    Install and secure the windshield washer hose assembly to the cowl panel.
  2. NOTE:

    Repeat the step for other side.

    Install and secure the heated windshield washer jet into the cowl panel.
  3. Connect the windshield washer hose to the 2 heated windshield washer jets.
    • Install the cowl panel.
    • Install the 12 clips.
  5. NOTE:

    Repeat the step for other side.

    Connect the electrical connector to the heated windshield washer jet.
  6. NOTE:

    Repeat the step for other side.

    Install the access cover.
  7. Connect the windshield washer hose.
    • Reposition the seals.
    • Install the 2 clips.
  9. Install the windshield wiper pivot arm.
    Refer to: Windshield Wiper Pivot Arm (501-16 Wipers and Washers, Removal and Installation).
