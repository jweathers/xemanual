Rear Door Lower Finisher (G2246176)

REMOVAL AND INSTALLATION

Loading...

Removal
NOTES:

  • This procedure contains some variation in the illustrations depending on the vehicle specification, but the essential information is always correct.

  • This procedure contains illustrations showing certain components removed to provide extra clarity.


  1. Lower the rear window.
  2. Release the rear upper door frame seal.
  3. CAUTION:

    Use a suitable trim tool.

    Remove the upper door trim panel.
  4. Remove the upper door finisher retaining screws.
  5. CAUTION:

    Use a suitable trim tool.

    Remove the upper door finisher.
  6. Remove the lower finisher retaining screw.
  7. CAUTION:

    Use a suitable trim tool.

    Remove the lower door finisher.
Installation
    • Install the rear lower door finisher.
    • Install the lower finisher retaining screws.
    • Install the upper door finisher retaining screws.
  3. Install the upper door trim panel.
  4. Install the upper door frame seal.
