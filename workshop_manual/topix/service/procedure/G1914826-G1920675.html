Removal
NOTE:
  • This procedure contains some variation in the illustrations depending on the vehicle specification, but the essential information is always correct.
  • This procedure contains illustrations showing certain components removed to provide extra clarity.

  1. Raise and support the vehicle on a suitable 2 post lift.
    Refer to: Lifting (100-02 Jacking and Lifting, Description and Operation).
  2. Drain the front differential.
    Refer to: Differential Draining and Filling (205-03 Front Drive Axle/Differential, General Procedures).
  3. Remove the front driveshaft.
    Refer to: Front Driveshaft (205-01 Driveshaft, Removal and Installation).
  4. Using a suitable tool, remove the pinion sleeve.
  5. Using a center punch and hammer, pierce a hole in the seal at the 12 O'clock position.
  6. WARNING:

    Be prepared to collect escaping oil.

    Screw a 2.5 x 15 mm self-tapping screw, a maximum of 2 turns into the input shaft seal and remove the seal using a pair of pliers.
Installation
  1. Clean the seal register and thoroughly inspect for damage.
  2. CAUTION:

    Make sure the seal protector sleeve fully covers the ridge on the pinion shaft.

    Install the pinion seal protector sleeve over the pinion shaft.
  3. CAUTION:

    This step must be carried out by hand. Failure to follow this instruction may result in damage to the pinion seal.

    Carefully install the new pinion seal.
    Renew Part: Front differential pinion repair kit Quantity: 1.
  4. Using the special tools, fully install the pinion shaft seal.
    Special Tool(s): JLR-205-1029, 307-520
  5. Remove the special tools.
  6. Carefully remove the protector sleeve.
  7. Using the special tool, install the cover.
    Special Tool(s): JLR-205-1029
  8. Install the front driveshaft.
    Refer to: Front Driveshaft (205-01 Driveshaft, Removal and Installation).
  9. CAUTION:

    Make sure the correct specification and quantity of oil is used.

    Fill the front differential.
    Refer to: Differential Draining and Filling (205-03 Front Drive Axle/Differential, General Procedures).
