Rear Wheelhouse Outer (G1785002)

REMOVAL AND INSTALLATION

Removal
NOTE:

In this procedure the rear wheelhouse outer is installed in conjunction with:

  • Rear bumper
  • Side panel
  • Quarter panel
  • Quarter panel lower extension
  • Diesel exhaust fluid tank
  • Fuel tank filler pipe
  1. NOTE:

    The rear wheelhouse outer is detached from the inner quarter panel assembly service panel.

    The rear wheelhouse outer is serviced as indicated.
  2. Before commencing this procedure make sure that you are aware of all Health and Safety requirements.For additional information, refer to: Health and Safety Precautions (100-00, Description and Operation).
  3. For further information on the methods, tools and fixings used in this procedure refer to the body repairs - general information section.
    4. NOTES:

    • Where breakstem fasteners (BSF) removal is instructed remove fastener shaft with a 4mm punch and remove the head with a 6.5mm drill bit. Clean all the drilled holes to make sure they are free from debris.

    • Where self piercing rivets (SPR's) removal is instructed use either the approved SPR tool or 5.3mm drill bit.

    • Any joints sealed as part of vehicle manufacture must be sealed as part of the repair.

  4. For additional information, refer to: Rear Bumper (501-19, Removal and Installation).
  5. For additional information, refer to: Side Panel (501-29 Side Panel Sheet Metal Repairs, Removal and Installation).
  6. For additional information, refer to: Quarter Panel (501-30 Rear End Sheet Metal Repairs, Removal and Installation).
  7. For additional information, refer to: Quarter Panel Lower Extension (501-30 Rear End Sheet Metal Repairs, Removal and Installation).
  8. NOTE:

    Left hand only.

    For additional information, refer to: Diesel Exhaust Fluid Tank (303-08A, Removal and Installation).
  9. NOTE:

    Right hand only.

    For additional information, refer to: Fuel Tank Filler Pipe (310-01A, Removal and Installation).
  10. Remove the SPR's as indicated.
  11. Remove the SPR's as indicated.
  12. Remove the SPR's and the bolts as indicated.
  13. Remove the SPR's as indicated.
  14. Carefully separate the joints and remove the rear wheelhouse outer.
Installation
CAUTION:

Where structural adhesive is to be applied between surfaces it is essential compression is applied to the joints during the curing period.

NOTES:

  • BSF installation involves using the Jaguar approved BSF installation tool and requires a hole diameter of 6.5mm.

  • SPR installation requires using the Jaguar approved SPR tool.

  1. Remove the rivet remnants.
  2. Dress flanges where necessary.
  3. Detach the rear wheelhouse outer from the inner quarter panel assembly as indicated.
  4. NOTE:

    Left hand only.

    Check and make sure that the diesel exhaust fluid tank brackets are correctly installed. If any fixings are not installed follow the next steps.
  5. NOTE:

    Left hand only.

    Drill holes where the BSF's are to be installed as indicated.
  6. NOTES:

    • Left hand only.

    • Make sure that all installed BSF's are sealed with a Jaguar recommended sealer.

    Install the BSF's as indicated.
  7. Offer up the new rear wheelhouse outer and clamp into position. Check alignment if correct proceed to next step, if not rectify and recheck before proceeding.
  8. Drill holes where the BSF's are to be installed as indicated.
  9. Drill holes where the BSF's are to be installed as indicated.
  10. Drill holes where the BSF's are to be installed as indicated.
  11. Drill holes where the BSF's are to be installed as indicated.
  12. Remove the rear wheelhouse outer.
  13. Deburr the drilled holes.
  14. Using a fine bristle disc clean and prepare the panel surfaces.
  15. Apply the coupling agent where the Jaguar recommended bonding material is to be applied and allow to dry.
  16. NOTE:

    Make sure a continuous bead of adhesive surrounds fixing holes.

    Apply a 5mm zig zag bead of the Jaguar recommended bonding material as indicated.
  17. Offer up the new rear wheelhouse outer and clamp into position. Check alignment if correct proceed to next step, if not rectify and recheck before proceeding.
  18. Install the BSF's as indicated.
  19. Install the BSF's and the SPR as indicated.
  20. Install the BSF's and the bolts as indicated and tighten the bolts to 25Nm.
  21. Install the BSF's as indicated.
  22. Remove any excess adhesive.
  23. Make sure that any open or exposed panel joints are correctly sealed.
  24. Make sure that all installed SPR's are sealed using a Jaguar recommended sealer.
  25. Make sure corrosion protection is applied to all areas affected by repair.
  26. The installation of associated panels and components is the reversal of the removal procedure.
