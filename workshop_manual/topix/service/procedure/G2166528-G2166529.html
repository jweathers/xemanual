Removal
  1. Remove the front door trim panel for access only.
    Refer to: Front Door Trim Panel (501-05 Interior Trim and Ornamentation, Removal and Installation).
  2. Disconnect the electrical connector.
    • Release the mid-range speaker connector and wiring harness.
    • Remove the 5 screws.
    • Release the 3 retaining clips.
    • Remove the mid-range speaker trim panel.
    • Remove the 3 screws.
    • Remove the mid-range speaker.
Installation
    • Install the mid-range speaker.
    • Install the 3 screws.
      Torque: 2.5 Nm
    • Install the mid-range speaker trim panel.
    • Make sure the 3 retaining clips are secure.
    • Install the 5 screws.
    • Install the mid-range speaker connector and wiring harness.
  3. Connect the front door trim panel wiring harness electrical connector to the mid-range speaker wiring harness connector.
  4. Install the front door trim panel.
    Refer to: Front Door Trim Panel (501-05 Interior Trim and Ornamentation, Removal and Installation).
