B-Pillar Inner Panel (G1784996)

REMOVAL AND INSTALLATION

Removal
NOTE:

In this procedure the B-pillar inner panel is installed in conjunction with:

  • Front fender
  • Front door
  • Rear door
  • Front seat
  • Rear seat cushion
  • Rocker panel and B-pillar outer panel
  • B-pillar outer panel
  • B-pillar reinforcement
  1. The B-pillar inner panel is serviced as indicated.
  2. Before commencing this procedure make sure that you are aware of all Health and Safety requirements.For additional information, refer to: Health and Safety Precautions (100-00 General Information, Description and Operation).
  3. For further information on the methods, tools and fixings used in this procedure refer to the body repairs - general information section.
    4. NOTES:

    • Where breakstem fasteners (BSF) removal is instructed remove fastener shaft with a 4mm punch and remove the head with a 6.5mm drill bit. Clean all the drilled holes to make sure they are free from debris.

    • Where self piercing rivets (SPR's) removal is instructed use either the approved SPR tool or 5.3mm drill bit.

    • Any joints sealed as part of vehicle manufacture must be sealed as part of the repair.

  4. For additional information, refer to: Fender (501-02 Front End Body Panels, Removal and Installation).
  5. For additional information, refer to: Front Door (501-03 Body Closures, Removal and Installation).
  6. For additional information, refer to: Rear Door (501-03 Body Closures, Removal and Installation).
  7. For additional information, refer to: Front Row Seat (501-10 Seating, Removal and Installation).
  8. For additional information, refer to: Rear Seat Cushion (501-10 Seating, Removal and Installation).
  9. For additional information, refer to: Rocker Panel and B-Pillar Outer Panel (501-29 Side Panel Sheet Metal Repairs, Removal and Installation).
  10. For additional information, refer to: B-Pillar Outer Panel (501-29 Side Panel Sheet Metal Repairs, Removal and Installation).
  11. For additional information, refer to: Side Panel Reinforcement Front Assembly (501-29 Side Panel Sheet Metal Repairs, Removal and Installation).
  12. Release and position the floor carpet to one side.
  13. Release and position the floor noise, vibration and harshness (NVH) material to one side.
  14. Release and position the wiring harness to one side.
  15. Remove the SPR's as indicated.
  16. Remove the SPR's as indicated.
  17. Using a suitable tool cut and remove the locating pins as indicated.
  18. Carefully separate the joints and remove the B-pillar inner panel.
Installation
CAUTION:

Where structural adhesive is to be applied between surfaces it is essential compression is applied to the joints during the curing period.

NOTES:

  • BSF installation involves using the Jaguar approved BSF installation tool and requires a hole diameter of 6.5mm.

  • SPR installation requires using the Jaguar approved SPR tool.

  1. Remove the rivet remnants.
  2. Dress the flanges where necessary.
  3. Check and make sure that the B-pillar inner bracket is correctly installed. If any fixings are not installed follow the next steps.
  4. Drill holes where the BSF's are to be installed as indicated.
  5. NOTE:

    Make sure that the BSF's are installed from the inside.

    Install the BSF's as indicated.
  6. Using a suitable tool remove the locating pins from the new B-pillar inner panel as indicated.
  7. NOTE:

    Make sure that M10 rivnuts are used.

    Install rivnuts to the new B-pillar inner panel as indicated.
  8. Offer up the new B-pillar inner panel and clamp into position. Check alignment if correct proceed to next step, If not rectify and recheck before proceeding.
  9. Drill holes where the BSF's are to be installed as indicated.
  10. Drill holes where the metal inert gas (MIG) plug welds are to be installed as indicated.
  11. Remove the B-pillar inner panel.
  12. Deburr the drilled holes.
  13. Using a fine bristle disc clean and prepare the panel surfaces.
  14. Apply the coupling agent where the Jaguar recommended bonding material is to be applied and allow to dry.
  15. NOTE:

    Make sure a continuous bead of adhesive surrounds fixing holes.

    Apply a 5mm bead of Jaguar approved bonding material to the new B-pillar inner panel as indicated.
  16. Offer up the new B-pillar inner panel and clamp into position. Check alignment if correct proceed to next step, If not rectify and recheck before proceeding.
  17. Install the BSF's and the SPR's as indicated.
  18. NOTE:

    Make sure that thread locking adhesive is applied to the studs and that they are installed to the correct depth.

    Install the threaded studs as indicated.
  19. Install the MIG plug welds as indicated.
  20. Dress all welds.
  21. Remove any excess adhesive.
  22. Make sure that any open or exposed panel joints are correctly sealed.
  23. Make sure that all installed SPR's are sealed using a Jaguar recommended sealer.
  24. Make sure corrosion protection is applied to all areas affected by repair.
  25. The installation of associated panels and components is the reversal of removal procedure.
