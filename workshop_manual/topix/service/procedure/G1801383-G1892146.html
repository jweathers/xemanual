Draining
CAUTION:

Residual pressure may still be present in the fuel system when disconnecting a fuel line. Make sure to cover the fuel line connection with a lint free cloth to capture any escaping fuel.

  1. Refer to: Petrol and Petrol-Ethanol Fuel Systems Health and Safety Precautions (100-00 General Information, Description and Operation).
  2. Remove the relay fuel pump mini fuse, located in the rear junction box.
  3. Remove the fuel filler cap.
  4. Start the engine and allow it to idle until the engine stalls.
Filling
  1. NOTE:

    Make sure all repairs have been carried out before proceeding to the following steps.

    Install the fuel pump fuse.
  2. Install the fuel filler cap.
  3. Crank the engine until it starts.
  4. Read and clear stored DTC fault codes.
