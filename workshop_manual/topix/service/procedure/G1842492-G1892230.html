Activation
CAUTION:

Failure to follow this instruction may result in damage to the component.

  1. Set the ignition to the on position, make sure the air conditioning (A/C) is in the off position.
  2. Start the engine and allow to run for a minimum of 5 minutes.
  3. Set the heater controls to 22°C, with the fan speed set to 75%.
  4. Switch on the A/C system.
  5. Open all air vents in the dashboard.
  6. Run the A/C system for a minimum of 5 minutes, while the engine is still at idle speed.
  7. Once this is achieved the compressor is stabilized, with the oil being distributed evenly throughout the system.
