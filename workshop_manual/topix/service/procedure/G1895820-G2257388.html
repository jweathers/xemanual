Removal
NOTE:
  • This procedure contains some variation in the illustrations depending on the vehicle specification, but the essential information is always correct.
  • This procedure contains illustrations showing certain components removed to provide extra clarity.

All vehicles

  1. Raise and support the vehicle on a suitable 2 post lift.
    Refer to: Lifting (100-02 Jacking and Lifting, Description and Operation).

Project 8 vehicles

  1. Remove the air deflector.
    Refer to: Engine Undershield (501-02 Front End Body Panels, Removal and Installation).

Vehicles with 3.0L diesel engine

  1. Remove the exhaust system.
    Refer to: Exhaust System - Vehicles Without: Diesel Exhaust Fluid (309-00B Exhaust System - INGENIUM I4 2.0L Diesel, Removal and Installation).
    Refer to: Exhaust System - Vehicles With: Diesel Exhaust Fluid (309-00B Exhaust System - INGENIUM I4 2.0L Diesel, Removal and Installation).

All vehicles

    • Mark the position of the front driveshaft in relation to the transmission flange.
    • Remove and discard the 4 bolts.
  2. Remove the front driveshaft.
Installation

All vehicles

  1. NOTE:

    Apply the required quantity of grease meeting the correct specification.

    Install the front driveshaft.
    Refer to: Specifications (205-01 Driveshaft, Specifications).
  2. NOTE:

    Make sure that you re-align the driveshaft to the transmission flange using the alignment mark.

    Install the 4 new bolts.
    Renew Part: Driveshaft to front output flange bolts Quantity: 4.
    Torque:
     
    Stage 1: 40 Nm
    Stage 2: 90°

Vehicles with 3.0L diesel engine

  1. Install the exhaust system.
    Refer to: Exhaust System - Vehicles Without: Diesel Exhaust Fluid (309-00B Exhaust System - INGENIUM I4 2.0L Diesel, Removal and Installation).
    Refer to: Exhaust System - Vehicles With: Diesel Exhaust Fluid (309-00B Exhaust System - INGENIUM I4 2.0L Diesel, Removal and Installation).

Project 8 vehicles

  1. Install the air deflector.
    Refer to: Engine Undershield (501-02 Front End Body Panels, Removal and Installation).
