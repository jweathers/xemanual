Removal
  1. CAUTION:

    If a new instrument cluster is to be installed, the diagnostic tool must be connected prior to removal, the data must then be downloaded from it and the keys (remote control handsets) set into default mode. Failure to follow this instruction will result in permanent damage to the keys.

    NOTE:

    If a new instrument cluster is to be installed, make sure that all keys (remote control handsets) are present.

    Configure the instrument cluster and ignition keys using the diagnostic tool.
  2. Refer to: Startup Battery Disconnect and Connect (414-01, General Procedures).
  3. Fully extend and lower the steering column for access.
  8. CAUTION:

    Take extra care not to damage the instrument cluster face.

Installation
  1. To install reverse the removal procedure.
  2. Configure the instrument cluster and ignition keys using the diagnostic tool.
